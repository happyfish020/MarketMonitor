DRS（否决） ＞ Gate（权限边界） ＞ Execution（摩擦/节奏）

【Gate 含义】
- ALLOW (A)：允许进攻
👉 - NORMAL (N)：结构正常，但不鼓励进攻
👉 - CAUTION：禁止加仓，只能防守或不动
- D / FREEZE：必须防守

【Execution（执行摩擦）】
- Execution band = D1：轻摩擦/偏中性（仍需服从 Gate/DRS）
- Execution band = D2：摩擦偏高（追价/频繁调仓胜率下降）
- Execution band = D3：摩擦很高/结构压力大（制度倾向去风险）
- Execution band = NA：数据不足（不影响 Gate/DRS）

【DRS（日度风险信号）】
👉 - GREEN：风险环境可控（不是进攻信号）
- YELLOW：需降档执行
- RED：否决一切进攻，仅允许防守

【CAUTION 状态下的冻结规则】
- 禁止：加仓、追高、放大试错
- 允许：维持仓位、减仓、防守性调整

【轻仓试错（严格定义）】
- 不增加总风险敞口
- 失败成本可忽略
- 仅用于验证，不用于进攻
👉 表示与当日制度状态直接相关的高亮提示（仅用于理解，不构成制度判断）。
